# Физика для программиста

Образовательный веб-сайт для изучения физики в контексте программирования игр.

## Описание проекта

Сайт создан для получения навыков в программировании с точки зрения физики. В современном мире программирование и физика становятся неотъемлемыми частями научного и технического прогресса. Наш сайт нацелен на то, чтобы помочь пользователям развить навыки в программировании, используя физику как основу для практического обучения.

## Функциональность

- **Теоретическая часть** - изучение законов физики в контексте игровой разработки
- **Материалы для разработчика** - полезные ресурсы, книги и ссылки
- **Обучение физике в играх** - видеоуроки по различным игровым движкам
- **Тестер знаний** - интерактивный тест с таймером и системой баллов
- **Физическая мини-игра** - практическое применение физических законов

## Технологии

- Python + Flask
- HTML5/CSS3
- JavaScript
- Адаптивный дизайн
- Анимированные фоны и эффекты
